# Getting Started

Two supported ways to run the Loaded Price Monitor — pick whichever suits you:

| Option | Best for | Cost | Setup time |
|---|---|---|---|
| **[Hosting on GitHub Actions →](GETTING_STARTED_GITHUB.md)** | Nothing to install. Runs in the cloud on GitHub's free CI. | Free (public repo) | ~5–10 min |
| **[Hosting locally →](GETTING_STARTED_LOCAL.md)** | Raspberry Pi, NAS, home server, or laptop. Maximum privacy, no GitHub involvement required. | Free, your hardware | ~10–20 min |

Both routes use the same `products.json` config, the same notification rules,
and the same Python / Rust / Go implementations. You can also switch between
them later, or run both side-by-side.

If you're not sure, start with **GitHub Actions** — it has nothing to install
and you can always move to local hosting later.
